#Yii2-Module-Testing

##Тестовое задание: 
создать модуль тестирования, который включаем в себя:

- таблицу тестов (поля: id теста, название теста, описание теста);
- таблицу вопросов (поля: id вопроса, заголовок вопроса, тип вопроса (один правильный вариант ответа, несколько вариантов ответов));
- связывающая таблица вопросов и тестов (id, id вопроса, id теста);
- таблицу ответов (id ответа, id вопроса, текст ответа, тип ответа (правильный/неправильный))

###Функционал

 1. Сделать возможность добавления/удаления/редактирования тестов;
 2. Сделать возможность добавления/удаления/редактирование вопросов с возможностью привязывания вопроса к нескольким тестам;
 3. Сделать возможность добавления/удаления/редактирования ответа с привязкой к определенному вопросу (можно сделать обычным выпадающим списком)
 4. На фронтенде сделать виджет, который будет отображать определенный тест (в качестве параметра виджета передаем id нужного теста), в зависимости от типа вопроса - ответы выводим в виде radiobutton или же checkbox
 5. Организовать прохождение теста с использованием ajax, т.е. форму отправляем аяксом и в зависимости от результата теста обычным alert'ом сообщаем пользователю результат тестирования (прошел его или провалил).
